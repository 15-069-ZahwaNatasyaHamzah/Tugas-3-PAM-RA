## Implementasi 

### 1. Kotlin Multiplatform
- Menggunakan Kotlin Multiplatform Project (KMP)
- Kode UI berada pada commonMain
- Dapat dijalankan pada beberapa platform seperti Android dan Desktop

### 2. Jetpack Compose Multiplatform
- Menggunakan Compose UI untuk membuat tampilan aplikasi
- Layout menggunakan Column, Row, dan Card
- Menggunakan komponen seperti Text, Image, Icon, dan Button

### 3. Resource Management
- Menggunakan composeResources
- Mengakses gambar menggunakan painterResource
- Menampilkan foto profil berbentuk circle

### 4. State Management
- Menggunakan remember dan mutableStateOf
- Mengontrol state dialog informasi
- Dialog muncul saat tombol Info ditekan
